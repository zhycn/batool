## 这本不该存在。

但世界太乱了，所以我们做了它。

你有没有试过，只是想打开 Notion，却在 Google 里点进三个广告、两篇软文、一个失效链接？ 

你有没有想过：为什么找一个工具，比用它还累？

够了。

Batool 是一张干净的桌子。  

上面只放你真正要用的东西——官网链接，仅此而已。

没有推荐算法，没有流量分成，没有“猜你喜欢”。

只有搜索框、图标、名字，和一个按钮：**直达**。

它快得像念头一样。  
你敲几个字母，按回车——人就到了官网。  
中间没有任何东西拦你。

我们把它做成了静态页面，因为信任不该依赖服务器。  
我们开源了它，因为你应该拥有自己的入口。

如果你也受够了噪音，  
欢迎回家。

→ [立即使用](https://batool-delta.vercel.app/)
